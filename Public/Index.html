Endpoint Documentation

1. Endpoint Beranda

HTTP Method: GET

Path: /

Deskripsi: Endpoint ini digunakan untuk menampilkan halaman beranda. Ketika mengakses root URL, server akan mengirimkan file HTML halaman beranda.

Contoh Penggunaan: POST http://localhost:3001/order

Body:
{ "id_product": 1, "nm_product": "Nama Produk", "price": 99 }

2. Endpoint Pemesanan Produk

HTTP Method: POST

Path: /order

Deskripsi: Endpoint ini digunakan untuk menerima data pemesanan produk. Data produk seperti ID, nama produk, dan harga akan dikirimkan dalam body request. Jika data tidak lengkap, akan merespons dengan status 400 (Bad Request).

Contoh Penggunaan:

HTTP Request: POST http://localhost:3001/order
Body:
{ "id_product": 1, "nm_product": "Nama Produk", "price": 99 }

3. Menjalankan Server

Deskripsi: Server ini akan berjalan pada alamat IP "192.168.56.1" dan port 3001. Informasi ini ditampilkan di console saat server dijalankan.
